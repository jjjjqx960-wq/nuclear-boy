## 边界

这里只描述稳定的小能力，不组合 Compose 布局，不依赖 ViewModel、Agent、文件系统或网络。

## 允许依赖

允许依赖 Kotlin 标准库和同层纯数据类型。

## 禁止事项

不要在零件层读取 Android Context、持久化文件、调用模型接口或直接渲染界面。

## 常用命令

- `./gradlew :ui-chat:compileDebugKotlin`
- `./gradlew :ui-chat:testDebugUnitTest`
