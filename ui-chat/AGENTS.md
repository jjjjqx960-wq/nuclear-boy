## 目录职责

`ui-chat` 是部件层模块，负责聊天界面、聊天 ViewModel、斜杠命令、消息渲染、Token HUD 和聊天侧工作区交互，包括文件浏览、过滤、过滤输入命中数量持续显示、无匹配清除过滤入口、摘要、排序、选择、全选/补选数量提示、只看/全部切换数量提示、图标操作数量语义、无已选可见总大小提示、当前可见直接引用、预览、引用入口、引用数量提示、单文件引用相对路径反馈、单文件引用大小反馈和引用后大小反馈。

## 边界

这里组合 Agent、模型、技能和文件组件形成用户聊天工作流，不直接实现底层模型协议或文件系统细节。

## 允许依赖

可以依赖 `common`、`api-deepseek`、`agent-core`、`skills`、`tools-docgen` 和 Compose UI 库。

## 禁止事项

不要在聊天日志或系统消息里输出明文秘密值。不要把持久化文件格式散落到多个 UI 组件中。

## 常用命令

- `./gradlew :ui-chat:test`
- `./gradlew :ui-chat:compileDebugKotlin`

## 验证方式

验证普通发送、`/goal`、`/loop`、`/stop`、`/compact`、`/rewind`、消息持久化、流式状态更新，以及文件面板浏览、过滤、过滤输入命中数量持续显示、无匹配清除过滤入口、摘要、排序、选择、全选/补选数量提示、只看/全部切换数量提示、图标操作数量语义、无已选可见总大小提示、当前可见直接引用、预览、引用数量提示、单文件引用相对路径反馈、单文件引用大小反馈、引用后大小反馈和引用链路。
