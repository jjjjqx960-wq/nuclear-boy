## 项目简介

Nuclear Boy 是一个基于 Android 的移动端 AI 编程助手，内置 Agent 引擎、工具注册、Python 执行器、三层记忆系统等能力。

## 技术栈

- **UI**: Jetpack Compose + Material 3 (暖色主题 #FF8C42)
- **DI**: Hilt (SingletonComponent)
- **网络**: OkHttp 4 + 流式 SSE
- **数据库**: Room (SQLite)
- **Python**: Chaquopy 15 (嵌入式 CPython)
- **序列化**: kotlinx.serialization

## 构建

1. 安装 Android Studio + SDK 35
2. 设置 `ANDROID_HOME` 环境变量
3. `./gradlew assembleDebug`

## 1.0.6 重点

- 文件面板新增当前目录快速过滤，可按文件名、扩展名或路径快速缩小列表。
- 过滤输入显示当前目录数量，输入后支持一键清除，便于继续浏览。
- 过滤能力下沉到零件层，搜索输入下沉到组件层，文件面板只负责组合和导航。

## 1.0.5 重点

- 长对话新增智能跟随滚动：在底部时继续跟随流式回复，上滑阅读历史时不再被强制拉回底部。
- 新增“到底部”浮动入口，阅读历史后可一键回到最新消息。
- 滚动判断下沉到零件层，浮动入口下沉到组件层，聊天屏幕只负责组合状态和用户意图。

## 1.0.4 重点

- 文件面板新增“引用到输入”，可一键把项目文件带入聊天草稿，减少手打路径和路径写错。
- 文件预览弹窗也支持引用，浏览、确认、提问之间的衔接更自然。
- 文件引用会自动转成项目相对路径，更贴合 Agent 文件工具的使用方式。

## 1.0.3 重点

- 聊天输入区新增命令快捷栏，常用 `/goal`、`/loop`、`/compact`、`/rewind`、`/model` 和停止任务入口可一键触达。
- 附件入口显示当前项目文件数量角标，减少打开文件面板前的信息盲区。
- 按零件/组件/部件分层补齐聊天快捷命令目录说明和更新日志。

## 1.0.2 重点

- 修复第三方 OpenAI 兼容服务的地址归一化与无鉴权网关支持。
- 第三方模型支持列表化管理，可新增、点选、删除，并可在主聊天页左上角快速切换。
- 设置页新增第三方模型测试诊断，可提示鉴权、路径、模型名、网络超时和 HTTP 私有网关问题。
- 修复流式工具调用的多工具 index 累积问题。
- 移除 API 客户端默认信任所有 TLS 证书的高风险行为。
- 持久化 `/goal` 会话目标，并修复 `/loop` 运行中无法稳定 `/stop` 的问题。
- Python 工具执行链调整为非隔离执行，设置页与命令帮助中移除沙箱开关。
- 对齐 Gradle wrapper 到本地已验证的 8.14.4。
