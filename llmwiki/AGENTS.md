## 目录职责

`llmwiki` 是核弹男孩项目的 LLM 可读知识索引，负责提供模块树、关键链路、常用命令和维护边界。

## 边界

这里只存放面向 LLM/Agent 的项目说明，不承载源码实现、构建产物、密钥、用户数据或运行日志。

## 允许依赖

内容可以引用仓库内模块、源码路径和验证命令。引用必须保持脱敏，不复述 API Key、Token 或个人数据。

## 禁止事项

不要把聊天记录、私有接口凭据、ADB 设备个人信息、构建日志全文或模型响应正文写入本目录。

## 常用命令

- `./gradlew :app:assembleDebug`
- `./gradlew test`
- `rg -n "TODO|FIXME" .`

## 验证方式

更新后检查路径是否仍存在，确认命令与当前 Gradle 模块一致，并运行敏感信息扫描。
